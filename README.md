🌞🌙 SOL Y LUNA ERP
Sistema de gestión empresarial (ERP) desarrollado en Python/Flask, diseñado para pequeñas y medianas empresas en Costa Rica. Incluye facturación electrónica compatible con los requisitos del Ministerio de Hacienda.
🚀 Módulos disponibles

Inventario — Gestión de productos, stock, kardex y alertas
Facturación — Facturación electrónica con códigos CABYS y actividades ATV
CXC — Cuentas por cobrar y recibos de pago
CXP- Cuentas por pagar
Panel Admin — Gestión de empresas y usuarios del sistema

🛠️ Tecnologías

Python 3.x / Flask
SQLite (dos bases de datos independientes)

⚙️ Instalación
bashgit clone https://github.com/tu-usuario/erp-sol-y-luna.git
cd erp-sol-y-luna
pip install -r requirements.txt
python scripts/crear_db.py
python scripts/cargar_actividades.py
python scripts/cargar_cabys.py
python app.py
